# NTU EDU Bot - 教育型智慧聊天機器人


##### 發想動機-學生日常生活的困擾
* 忘記要點名或小考作業等
* 每天最難的問題-等一下要吃什麼
* 這堂課，可以修嗎？

> 何不設計一款Bot，可以通知我課程資訊
> 又可以當作NTU學生專屬的Virtual Assistant!

##### 基本功能
1. 課程訂閱
2. 美食推薦
3. GPA 預測
